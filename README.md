# react support
- 一个`react`模板快速生成的vscode插件
- 目前支持`class component`和`function component`以及`hook`书写方式
- 快捷方式
  - `react`模块
    - `react class`： 生成`class component` 模板
    - `react func`： 生成`function component` 模板
    - `react hook`： 生成`react hook` 模板
  - js常用快捷键
    - `clg`: `console.log()`
    - `clw`: `console.warn()`
    - `cle`: `console.log()`
  - 安装方式：
    - 暂时只支持通过vsix来进行安装
- 后续将会支持更多的模板
